Photography Club Landing Page

A Tailwind CSS-based landing page created for a fictional photography club, focusing on design layout and event showcase.

Project Preview

Project Overview

This landing page was designed to mimic a real-world community site. It features:

  • A clean hero section with strong typography and a CTA button
  • Responsive card layout for upcoming events
  • Interactive calendar component styled with Tailwind
  • A gallery section with hoverable images
  • A grid-based footer with useful links and contact details

Technologies Used

Tailwind HTML CSS JavaScript

Challenges & Learnings

Creating a responsive layout with Tailwind was straightforward, but aligning the calendar and event cards took refinement. This project helped improve my understanding of utility-first CSS and component structure.
